# Texas Roleplay - Roblox Server

A large-scale roleplay server for Roblox featuring a county-sized map of Texas.

## Project Structure

- `src/` - Server scripts and game logic
- `src/Map/` - Map generation and terrain
- `src/Players/` - Player management and spawning
- `src/Server.lua` - Main server entry point

## Features

- County-sized explorable map
- Player spawning system
- Server-side game logic
- Scalable architecture

## Setup

1. Open the main place file in Roblox Studio
2. Add this folder as a Git repository
3. Scripts should be placed in ServerScriptService
4. Map assets should be in Workspace

## Development

This project uses Lua scripting for Roblox game development. All server scripts follow Roblox best practices.
